Mashinani League Cup: Custom Fantasy League Tracker

Developed a full-stack application to track and manage the rules, payments and leaderboard for a custom Fantasy Premier League mini-league. Features complex data handling and a dedicated Admin panel.

Full-Stack Backend Logic Data Fetching Leaderboard Payments Tracking

Complex Data Handling and Custom Rules

The core challenge of this project was integrating external data (FPL scores) with custom league rules, such as calculating 'Net Points' based on transfers and managing the 'Winner Takes All' payout structure shown in the UI. This required robust API consumption and server-side processing to accurately update scores per Game Week.

The system features a multi-view Leaderboard (By Net Points, By Wins, Gameweek View) and tracks Payments to maintain transparency. The segregated Admin section allows for manual adjustments, payment reconciliation and rule changes, demonstrating strong user role and permission management skills.

The dark-themed, data-rich interface was designed to make complex league information easy to digest, with clear visual hierarchy for the weekly winner and league rules.
